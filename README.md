G'AIM'E Light Gun – House of the Dead 2 button remap for the G'AIM'E light Gun.
You will need AutoHotkey v2 for this script to function. 

🎮 Features

* Plug & Play (No extra drivers needed)
* Full button remap (Nagivate and pause the game all from the gun)
* Automatic reload (Shot counter that will auto reload after 6 shots have been fired)

📦 Requirements

* AutoHotkey v2 - https://www.autohotkey.com/v2/
* House of the Dead 2 PC - I recommend: https://www.myabandonware.com/game/the-house-of-the-dead-2-beg
* G'AIM'E light gun

🚀 Installation

1. Install AutoHotkey v2
2. Download the `HOTD2Remap.ahk` file from this repo
3. Double-click the script to run it
4. Start HOTD2

🔫 Button layout

The G'AIM'E light gun simulates both keyboard and mouse - Below are the native button outputs from the gun to the keyboard, and what key AutoHotkey simulates.  
(See Picture in repo)

GUN > > >  Keyboard > > > AutoHotkey  > > > HOTD2

Trigger >  Left Click   > Left Click      > Shoot
A       >  Space Bar    > Right Click     > Reload
B       >  B            > Enter           > Pause / Start 
Coin    >  C            > Up Arrow        > Nagivate Up
Paus    >  Q            > Down Arrow      > Nagivate Down

ℹ️ Notes

To ensure best accuracy, I recommend plugging your G'AIM'E console in first and calibrating the gun to the screen you plan on playing HOTD2 from.
